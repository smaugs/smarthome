Sample Plugins
==============


This section shows sample plugins. There is a standard version and a version for plugins 
with a web interface.

Additionally there is an old sample plugin, in case you need backward comptibility to
SmartHomeNG versions v1.1 up tp v1.3.
 
The complete plugins with all files (specially the plugin with web interface has some additional files) can
be found on github (www.github.com/smarthomeng/smarthome in the /dev folder.
